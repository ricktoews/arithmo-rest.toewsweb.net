Decimal Calculator

http://api.arithmophile.com/dc/[denominator](/[numerator])

Specify denominator and optional numerator

Sequential array of decimal data for each fraction having the specified [denominator]. If the optional [numerator] is specified, the array will contain a single element. Each element in the array contains:
fraction The current fraction, written out as "[numerator] / [denominator]". If the fraction can be simplified, the simplified form is also provided.
decimal The decimal expansion of the fraction.
length The length of the decimal expansion.
repeating The number of digits in the decimal expansion that repeat, if any.
Examples
  • http://api.arithmophile.com/dc/7 - Returns decimals having a denominator of 7.
  • http://api.arithmophile.com/dc/49/5 - Returns decimal for fraction 5/49.

Triangular Numbers

http://api.arithmophile.com/tri/upto/[nth]

List the triangular numbers, up to the nth. (e.g., If nth is 3, list includes 1, 3, 6.)

Associative array of triangular numbers, with each key being the ordinal position of the triangular number.
[n] The nth triangular number
Examples
  • http://api.arithmophile.com/tri/upto/5 - Returns triangular numbers 1, 3, 6, 10, 15.

http://api.arithmophile.com/tri/[from_nth]/[to_nth]

List the triangular numbers within the specified range.

Associative array of triangular numbers, with each key being the ordinal position of the triangular number.
[n] The nth triangular number
Examples
  • http://api.arithmophile.com/tri/3/5 - Returns triangular numbers 6, 10, 15.

http://api.arithmophile.com/tri/test/[n]

Check the specified number to determine whether it is triangular.

number The number being tested for triangularity.
is_triangular "Yes" or "No", depending on whether [n] is triangular or not.
nth_triangular_number The ordinal position in the series of triangular numbers. (Provided only if [n] is a triangular number.)
Examples
  • http://api.arithmophile.com/tri/test/10
  • http://api.arithmophile.com/tri/test/12
Golden Ratio

http://api.arithmophile.com/phi/[n]

Returns the number phi, to the specified number of places after the decimal.

digits The number of phi digits returned.
phi_digits The first [n] digits to the right of the decimal of phi. These are grouped in space-delimited clusters of 10 digits each.
Examples
  • http://api.arithmophile.com/phi/100 Returns phi to 100 decimal places.

http://api.arithmophile.com/phi/powers(/[maximum power])

Returns the powers of phi, up to the indicated maximum.

Sequential array, each row of which contains:
power Power of phi for current row
phi_num Value of phi to [power], expressed as a fraction, with denominator of 2. (This is by design, even though it means that, for powers divisible by 3, the fraction could be further simplified.)
real_value Value of phi to [power], expressed as a real number approximation.
sqrt_5 The number multiplied by the square root of 5 in this power of phi, and with a denominator of 2.
whole The integer portion of this power of phi, given a denominator of 2.
denom The denominator of this power of phi. Currently, this is always 2.
Examples
  • http://api.arithmophile.com/phi/powers/5 Returns the powers of phi, up to phi^5.
Prime Numbers

http://api.arithmophile.com/p/upto/[maximum]

Returns a list of the prime numbers less than or equal to specified maximum.

primes_listed How many prime numbers there are between 1 and [maximum]
prime_numbers Associative array containing the prime numbers from 1 to [maximum]. Key is the ordinal position of the prime number.
Examples
  • http://api.arithmophile.com/p/upto/100 Returns the prime numbers less than or equal to 100.

http://api.arithmophile.com/p/[a]/[b]

Returns a list of the prime numbers between the specified integers.

primes_listed How many prime numbers there are between [a] and [b]
prime_numbers Associative array containing the prime numbers from [a] to [b]. Key is the absolute ordinal position of the prime number; e.g., the key for 101 would always be 26, since 101 is the 26th prime number.
Examples
  • http://api.arithmophile.com/p/10/20 Returns the prime numbers between 10 an 20, which are 11, 13, 17, 19.

http://api.arithmophile.com/p/[number]

Indicate whether the number is prime. Return its factors if composite.

number The number that was tested for primeness.
prime_factors Sequential array containing the unique prime factors of [number].
Examples
  • http://api.arithmophile.com/p/10 Returns 2 and 5 as the prime factors of 10.
  • http://api.arithmophile.com/p/5 Indicates that 5 is a prime number.
